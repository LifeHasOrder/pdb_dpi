DPI Calculator

Per-Atom Coordinate Precision for Protein Crystal Structures

The Diffraction Precision Index (DPI) estimates the coordinate uncertainty σ(x) for each atom in a crystal structure, based on the crystallographic R-factor, resolution, data completeness, and the number of observations and refined parameters.

Overall DPI is computed from Cruickshank (1999) Eqs. 24 & 31. Per-atom precision is scaled by √(Bi / Bavg) per Helliwell (2023) Eq. 2. See references below.

All computation happens entirely in your browser via Pyodide. No data is uploaded to any server.
